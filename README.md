# TaskFlow

![TaskFlow Branding](https://via.placeholder.com/1200x600.png?text=TaskFlow+Dashboard)

**TaskFlow** is a modern, full-stack task management application designed for clarity and productivity. Built with a robust **FastAPI** backend and a responsive **React** frontend, it features a professional SaaS-like UI, secure authentication, and real-time task tracking.

## üöÄ Features

-   **Professional Dashboard**:
    -   **Stats Overview**: Track visual metrics like *Total Tasks*, *Completion Rate*, and *Pending* items.
    -   **Task Management**: Create, edit, delete, and filter tasks with ease.
    -   **Optimistic UI**: Instant feedback on task updates.
-   **Modern Experience**:
    -   **Split-Screen Auth**: Beautiful, branded Login and Signup pages.
    -   **Responsive Design**: Fully optimized for desktop and mobile using **TailwindCSS**.
    -   **Interactive Elements**: Hover effects, smooth transitions, and glassmorphism details.
-   **Security**:
    -   **JWT Authentication**: Secure stateless authentication.
    -   **Password Hashing**: Industry-standard Argon2 hashing.

## üì¶ Installation & Setup

### Prerequisites
-   Node.js 16+
-   Python 3.8+

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/taskflow.git
cd taskflow
```

### 2. Backend Setup
Navigate to the backend directory and set up the Python environment.
```bash
cd backend
python -m venv venv
# Windows
.\venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

pip install -r requirements.txt
```

**Configuration**:
Create a `.env` file in `backend/`:
```ini
DATABASE_URL=sqlite:///./task.db
SECRET_KEY=your_super_secret_key_here
```

**Run Server**:
```bash
uvicorn main:app --reload
# Server running at http://localhost:8000
```

### 3. Frontend Setup
Navigate to the frontend directory.
```bash
cd ../frontend
npm install
```

**Run Client**:
```bash
npm run dev
# App running at http://localhost:5173
```
